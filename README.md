# 《市长模拟器》V1.0 (Mayor Simulator)

一款单局制城市政策决策模拟网页游戏。玩家在 20 个季度 (共 5 年) 内执掌架空城市“临州”，通过选择政策、控制投入力度、管理财政债务与处理突发事件，完成任期并交出城市答卷。

## 🚀 快速启动

```bash
# 1. 安装依赖
npm install

# 2. 启动开发服务器
npm run dev

# 3. 运行规则与 100 局数值平衡测试
npm run test

# 4. 构建生产环境包
npm run build

# 5. 预览生产构建
npm run preview
```

## 🎮 游戏核心玩法

1. **财政与债务**: 初始财政 48 亿，债务 32 亿。季度开销先消耗可用财政，不足部分转为债务。**债务超过 120 亿**立即触发“财政托管”失败结局。
2. **民心警报**: 民心低于 10 时记录警报，**连续两个季度低于 10** 立即触发“黯然离任”失败结局。
3. **三种投入力度**:
   - **试点 (0.6x 成本)**: 风险低，适合观察效果。
   - **全市推行 (1.0x 成本)**: 标准方案。
   - **攻坚投入 (1.5x 成本)**: 效果提升，建设周期减少 1 季度。
4. **政策与事件联动**:
   - 包含 16 项政策、6 种年度趋势、30 个城市突发事件与 8 种结局。
   - 解锁交通+工业、人才+住房、夜市+公交、环保+工业等 6 组强力联动效应。
5. **固定种子复盘 (PRNG)**:
   - 访问 `http://localhost:3000/?seed=12345` 即可使用特定种子完全复现随机数轨迹。
