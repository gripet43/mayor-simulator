# 安澜市城市模拟器核心指标字典 (DATA_DICTIONARY)

模拟器中包含的各维度核心指标如下表所示：

| 指标 (Metric) | 含义 (Meaning) | 单位 (Unit) | 正常范围 (Range) | 派生 (Derived) | 主要影响源 (Main Influences) |
|---|---|---|---|:---:|---|
| **GDP** | 全市季度生产总值 | 当季亿元 | 800 - 2000 | 否 | 6大产业产出合计 |
| **失业率** | 全市劳动年龄人口失业率 | % | 0 - 100 | 是 | 劳动力人口总数、各产业实际岗位容量 |
| **财政现金** | 市财政可用现金储备 | 亿元 | >= 0 | 否 | 税收、上级支付、融资到账、各项财政支出 |
| **债务余额** | 地方政府未偿还债务总和 | 亿元 | 0 - 1500 | 否 | 初始债务、新增贷款、每季度等额本金偿还 |
| **偿债压力** | 债务本息占季度可用财政收入比重 | % | 0 - 100 | 是 | 季度到期本金及利息、可用税收与非税收入 |
| **历史欠款** | 财政无法足额支付积累的拖欠金额 | 亿元 | >= 0 | 否 | 财政支出缺口 |
| **政府公信力** | 政府公共信誉度与执行力表现 | 0 - 100 | 0 - 100 | 否 | 年度目标达成率、欠款、超载、居民平均满意度 |
| **实际行政能力**| 季度可调配的政务处理资源上限 | 点数 | 8 - 16 | 是 | 基础值 12 点、行政效率修正 |
| **租金指数** | 片区平均租赁住房费用水准 | 基期100 | 60 - 200 | 否 | 人口流入、片区住房供需比、平均工资增速 |
| **房价指数** | 片区平均商品住房交易水准 | 基期100 | 60 - 250 | 否 | 租金水准、空置率、商业信心、供地政策 |
| **平均通勤时间**| 居民往返通勤消耗的平均分钟数 | 分钟 | 15 - 65 | 是 | 片区道路条件、交通可达性、拥堵指数 |
| **教育综合得分**| 教育公共服务的可及性与教学水准 | 0 - 100 | 20 - 95 | 是 | 教师充足率、学位学生比、教育财政预算充足率 |
| **医疗综合得分**| 医疗服务的充足率与综合急救能力 | 0 - 100 | 20 - 95 | 是 | 医疗预算投入、医护充足率、片区老龄化程度 |
| **环境质量得分**| 城市空气质量与生态水准 | 0 - 100 | 0 - 100 | 否 | 工业与物流废气排放强度、环保预算投入 |
| **洪涝风险指数**| 该季度洪涝灾害暴露风险 | 0 - 100 | 0 - 100 | 是 | 排水防洪管线能力、沿江防洪政策、季度降水 |
| **群体感受** | 六类群体（青年、家庭等）的感受值 | 0 - 100 | 0 - 100 | 是 | 9大民生维度指标的加权加和、临时政策修正 |
